# 🚦 Google Traffic Bot - Railway Edition

This is a web-based version of Google Traffic Bot optimized for deployment on Railway.

## 🌐 Live Deployment on Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/yourusername/google-traffic-bot)

## ✨ Features

- 🌐 **Web Interface**: Access via browser instead of desktop app
- 📋 **Proxy Management**: Upload proxies directly via web interface
- 🚀 **Headless Operation**: Runs without display on Railway servers
- 📊 **Real-time Logs**: View bot activity in real-time
- 🔄 **Auto-Detect Proxy Formats**: Supports HTTP, SOCKS4, SOCKS5

## 🚀 Quick Deployment

1. Click the "Deploy on Railway" button above
2. Wait for build to complete
3. Open the Railway URL
4. Upload proxies and start botting!

## 🔧 Manual Setup

```bash
# Clone repository
git clone https://github.com/SoHeil-R/google-traffic-bot.git
cd google-traffic-bot

# Install dependencies
npm install

# Start server
npm start
